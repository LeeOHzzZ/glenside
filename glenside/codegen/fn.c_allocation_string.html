    prefix: &str, 
    name: &str, 
    shape: &[usize], 
    dtype: DType

Expand description

Creates a representation (currently just a string with a C declaration) of an allocation, given the name, shape, dtype, and any prefix.

assert_eq!(
    glenside::codegen::c_allocation_string(
        "__my_prefix",
        "my_array",
        &[3, 4, 5],
        glenside::hw_design_language::DType::Fp32
    ),
    "__my_prefix float my_array[3][4][5];"
);
